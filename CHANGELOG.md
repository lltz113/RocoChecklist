# 更新日志

- 为侧边导航栏新增动态视觉效果（部分效果基于滚动监听），包括自动高亮当前清单项、颜色渐变变化等。
- 添加动态监听，使网页可根据当前版块自动更新 URL hash。（当前版本存在问题，待修复）

### 文档

- 变更侧边导航栏的部分样式。

## [0.3.0] (2026-02-22)

### 新增

- 新增侧边导航栏，可用于快速定位文档位置。

### 修复

- 修复“觉醒重生”和“圣域集（下）”版块清单项 ID 重复的 BUG，该 BUG 曾导致：
  清单页面无法独立保存上述各个清单项的勾选状态。

### 文档

- 变更文档的部分样式，使其适应新增的侧边导航栏。
- 修改部分文字描述，使其更加准确和生动。
- 修改标题行内容并居中。
- 新增网页图标。

## [0.2.0] (2026-02-21)

### 新增

- 新增四个分类统计版块：“勇者神祇”“四时邀约”“觉醒重生”“境启寻珍”。
- “哨兵因因的情报”版块新增“圣域集（下）”部分。
- 添加悬停提示信息功能：鼠标指针悬停在部分清单项时将显示详细信息。

### 文档

- 新增“关于”版块，罗列清单分类与统计的主要依据。
- 为每个版块对应的活动添加文字描述（如活动场景、地图编号、特殊说明等）。
- 统一清单的排版风格，使清单更整洁。
- 纠正个别错别字。

### 优化

- 使用`<label>`重写清单项。改动后，除点击方框外，点击方框后的文字也可实现勾选/取消勾选清单项操作。
- 在“哨兵因因的情报”版块中，“洛战复活”三期万象集的部分奖励被列入自动统计范围。
