# RocoChecklist

RocoChecklist 是一个简易网站，以清单的形式统计了游戏《洛克王国》中可以通过常驻活动免费获得的万象集宠物；此外，网站还可以根据宠物收集进度，计算可领取的图鉴奖励额度。  
该网站可供每位小洛克使用，记录自己的宠物获取进度，了解图鉴奖励的解锁情况。

---

## 如何使用

在地址栏输入 https://lltz113.github.io/RocoChecklist/ ，访问 RocoChecklist 网站。

仅需根据各位小洛克实际的宠物拥有情况，手动勾选清单项，网站便会自动统计并保存宠物的收集进度和奖励额度。

🚨 注意：清除浏览数据可能导致清单自动保存的进度丢失。

## 网站特性

RocoChecklist 具有以下特性：
- **操作便捷**：只需要鼠标单击清单项，便可执行勾选/取消勾选操作；此外，网页的侧边导航栏能够帮助小洛克快速定位清单项。
- **安全可靠**：作为项目站点，RocoChecklist 是静态网站，不会获取各位小洛克的个人信息；宠物收集进度保存在浏览器本地存储，不会上传至云端。
- **界面简洁**：网页无多余 UI 设计，内容简洁明了。
